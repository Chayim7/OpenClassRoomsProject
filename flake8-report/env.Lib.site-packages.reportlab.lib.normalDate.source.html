71 class NormalDate:
72 """
73 NormalDate is a specialized class to handle dates without
74 all the excess baggage (time zones, daylight savings, leap
75 seconds, etc.) of other date structures. The minimalist
76 strategy greatly simplifies its implementation and use.
77  
78 Internally, NormalDate is stored as an integer with values
79 in a discontinuous range of -99990101 to 99991231. The
80 integer value is used principally for storage and to simplify
81 the user interface. Internal calculations are performed by
82 a scalar based on Jan 1, 1900.

84 Valid NormalDate ranges include (-9999,1,1) B.C.E. through
