139 class Standard39(_Code39Base):
140 """
141 Options that may be passed to constructor:
142  
143 value (int, or numeric string required.):
144 The value to encode.
145  
146 barWidth (float, default .0075):
147 X-Dimension, or width of the smallest element
148 Minumum is .0075 inch (7.5 mils).
149  
150 ratio (float, default 2.2):
151 The ratio of wide elements to narrow elements.
152 Must be between 2.0 and 3.0 (or 2.2 and 3.0 if the
153 barWidth is greater than 20 mils (.02 inch))
154  
155 gap (float or None, default None):
156 width of intercharacter gap. None means "use barWidth".
157  
158 barHeight (float, see default below):
159 Height of the symbol. Default is the height of the two
160 bearer bars (if they exist) plus the greater of .25 inch
161 or .15 times the symbol's length.
162  
163 checksum (bool, default 1):
164 Wether to compute and include the check digit
165  
166 bearers (float, in units of barWidth. default 0):
167 Height of bearer bars (horizontal bars along the top and
168 bottom of the barcode). Default is 0 (no bearers).
169  
170 quiet (bool, default 1):
171 Wether to include quiet zones in the symbol.
172  
173 lquiet (float, see default below):
174 Quiet zone size to left of code, if quiet is true.
175 Default is the greater of .25 inch, or .15 times the symbol's
176 length.
177  
178 rquiet (float, defaults as above):
179 Quiet zone size to right left of code, if quiet is true.
180  
181 stop (bool, default 1):
182 Whether to include start/stop symbols.
183  
184 Sources of Information on Code 39:
185  
186 http://www.semiconductor.agilent.com/barcode/sg/Misc/code_39.html
187 http://www.adams1.com/pub/russadam/39code.html
188 http://www.barcodeman.com/c39_1.html
189  
190 Official Spec, "ANSI/AIM BC1-1995, USS" is available for US$45 from
191 http://www.aimglobal.org/aimstore/
192 """
